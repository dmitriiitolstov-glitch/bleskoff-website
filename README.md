# BleskOFF — редизайн сайта

Самостоятельная кодовая версия сайта детейлинг-студии BleskOFF. Проект собран на Astro, поддерживает админку Sanity и не связан с действующим сайтом в Tilda.

## Что уже работает

- адаптивная главная страница для компьютера и телефона;
- услуги, готовые программы, примеры работ, процесс, FAQ и контакты;
- переключение программ и мобильное меню;
- ссылки на телефон, WhatsApp, онлайн-запись и карту;
- локальные изображения без зависимости от Tilda CDN.
- редактирование услуг, акций, программ, FAQ и контактов через админку;
- SEO-метатеги, canonical, Open Graph, sitemap, robots.txt и разметка AutoWash;
- автоматическая проверка сборки в GitHub Actions.

## Запуск

Понадобится Node.js 22 или новее и pnpm.

```sh
pnpm install
pnpm dev
```

После запуска сайт будет доступен по адресу `http://localhost:4321`.

## Проверка и production-сборка

```sh
pnpm build
pnpm preview
```

Готовые статические файлы появятся в папке `dist`. Их можно разместить на обычном хостинге, Netlify, Vercel или Cloudflare Pages.

## Админка

Без подключения Sanity сайт использует безопасную локальную копию всего контента. Инструкция по включению редактора и выдаче клиентского доступа находится в [ADMIN_SETUP.md](./ADMIN_SETUP.md).

После подключения:

- `/admin` — понятная входная страница для клиента;
- `/studio/` — редактор содержимого;
- `pnpm cms:import` — загрузка стартового содержимого в Sanity.

## Основные файлы

- `src/pages/index.astro` — содержимое и структура главной;
- `src/styles/global.css` — вся визуальная система и адаптивность;
- `src/components/` — шапка и карточки услуг;
- `src/sanity/` — структура разделов админки;
- `src/content/defaults.ts` — резервное содержимое сайта;
- `src/lib/cms.ts` — получение опубликованного содержимого из Sanity;
- `public/images/` — изображения проекта.
